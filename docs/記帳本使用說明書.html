記帳本使用說明書

1. 設定頁：先建立你的記帳環境

第一次使用建議先到「設定」，把帳戶、類別與預算基礎資料整理好，之後新增記帳會更順。

銀行帳戶、信用卡、電子錢包、現金

可新增、刪除、排序付款方式。

部分帳戶可設定期初金額或標記是否列入可用金額。

收入類別

管理收入項目，例如工作薪資、獎金、加班費、股息、其他收入。

財務規劃

可建立規劃帳戶，例如緊急預備金、投資理財、定期定額。

設定初始金額後，統計頁會自動累計進出。

浮動支出類別

管理日常支出分類。

每個主類別底下可以設定子類別，例如：

  • 主食：早餐、午餐、晚餐
  • 交通費：捷運、高鐵、加油
  • 休閒娛樂：電影、遊戲、聚餐

固定支出

設定每月固定支出項目與金額，例如房租、保險、訂閱服務。

新增記帳時選到固定支出，App 會自動帶入設定金額。

貸款

設定貸款名稱與期初本金。

新增貸款還款記錄時，可填寫還本金，App 會自動計算利息與剩餘本金。

信用卡額度

設定各銀行信用卡額度。

統計頁會用來計算信用卡使用率。

維護與工具

可使用匯出 JSON 備份、匯入 JSON 備份、清除記帳資料、清空全部資料與設定、檢查更新、財務語言等功能。

介面顏色

可切換 App 主題色。

2. 開始使用

打開 App 後，請先登入帳號。

如果還沒有帳號，點選「註冊」建立帳號；如果忘記密碼，點選「忘記密碼？」寄送重設信。

登入後，資料會自動同步到雲端。畫面右上角可以看到同步狀態，也可以點「同步」手動同步。

3. 新增記帳

底部選單點選「新增」。

依序填寫：

  • 日期
  • 分類
  • 付款方式
  • 主類別
  • 子類別
  • 金額
  • 備註

支出請輸入負數，例如 -200；收入輸入正數，例如 50000。

填完後按「新增記帳」。

4. 信用卡分期

當付款方式選到信用卡時，會出現「不分期 / 分期」按鈕。

使用方式：

  1. 點選「分期」。
  2. 輸入期數，例如 6。
  3. App 會顯示每期攤還金額。
  4. 新增後，統計頁會依分期月份計算每月應攤金額。

在「統計 > 信用卡區」中，若該信用卡有分期項目，點擊該信用卡列可以展開分期明細。

5. 記錄管理

底部選單點選「記錄」。

可搜尋、篩選、編輯、複製或刪除記錄。

手機上可左滑記錄，顯示「複製」與「刪除」。

電腦上可用右鍵點選記錄，選擇「複製」或「刪除」。

6. 統計頁

底部選單點選「統計」。

統計頁可查看：

  • 本月支出
  • 本月收入
  • 可用金額
  • 可用現金
  • 月可用餘額
  • 年可用餘額
  • 收入、電子錢包、浮動支出、固定支出
  • 銀行、信用卡、財務規劃、貸款
  • 各類圖表

點擊「月可用餘額」或「年可用餘額」可查看明細。

7. 報表頁

底部選單點選「報表」。

報表分成：

  • 年度報表
  • 季 & 月報表

年度報表可查看預估、實際、差距、達成率、收入佔比與各項佔比。

切到單月時，可直接輸入各項預估金額，也可以按「帶入上月預估」。

8. 備份與還原

在「設定 > 維護與工具」中可使用：

  • 匯出 JSON 備份
  • 匯入 JSON 備份
  • 清除所有記帳資料
  • 清空全部資料與設定

大量修改前建議先匯出備份。

9. 財務語言

在「設定 > 維護與工具」中點選「財務語言」。

可以搜尋、展開說明，也可以自行新增、編輯或刪除詞彙。

10. 更新與離線使用

當 App 有新版本時，畫面上方會出現更新提示。

點選「立即更新」即可載入新版本。

App 支援離線使用。離線期間新增或修改的資料會先存在本機，恢復網路後可再手動同步。

11. 建議使用流程

  1. 登入或註冊帳號。
  2. 到「設定」建立銀行帳戶、信用卡、電子錢包、現金與類別。
  3. 設定固定支出、貸款、信用卡額度與財務規劃。
  4. 到「報表」的單月頁面輸入預估金額。
  5. 回到「新增」開始記帳。
  6. 定期到「統計」查看本月狀況。
  7. 每月檢查「報表」中的預估與實際差距。
  8. 定期匯出 JSON 備份。
